# 📚 LibraLog — NEU Library Logger

A real-time library visitor logging system for New Era University.

## Tech Stack
- **Frontend**: React 18 + React Router v6
- **Database**: Firebase Firestore (real-time)
- **Auth**: Firebase Authentication
- **Hosting**: Vercel
- **Font**: Poppins

## Features
- 🔐 Admin authentication
- 📋 Log visitor time-in and time-out
- 📊 Real-time dashboard with live stats
- 🔍 Search and filter visit logs
- 📈 Reports with date range and CSV export
- ⚙️ Settings (profile, password, add admins)

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Copy and fill in your Firebase config
cp .env.example .env.local

# 3. Start development server
npm start
```

See the PDF setup guide for complete Firebase + Vercel setup instructions.
